Java Console-Based Projects

This repository contains a collection of console-based Java applications developed to practice core programming concepts and problem-solving skills. These projects demonstrate the use of Java fundamentals, control structures, object-oriented programming concepts, and user interaction through the console.

Each project is designed with a menu-driven interface to simulate real-world applications and improve logical thinking.

Technologies Used

Java

Object-Oriented Programming (OOP)

Scanner Class for User Input

Conditional Statements

Loops

Basic Data Structures

Random Number Generation

1. ATM Mini System

A console-based simulation of an ATM machine that allows users to perform basic banking operations.

Features

Check account balance

Deposit money

Withdraw money

Menu-driven interaction

Concepts Used

Conditional statements

Loops

User input handling

2. Banking Management System

A simple banking system that allows users to manage basic banking operations through the console.

Features

Create a bank account

Deposit money

Withdraw money

View account details

Concepts Used

Object-Oriented Programming

Methods

Menu-driven program structure

3. Number Guessing Game

A console-based game where the user attempts to guess a randomly generated number.

Features

Random number generation

User input guesses

Hint messages (higher or lower)

Score or attempt tracking

Concepts Used

Random class

Loops

Conditional logic

4. Online Exam System

A basic quiz application that asks the user multiple questions and calculates the final score.

Features

Multiple-choice questions

Score calculation

Final result display

Concepts Used

Arrays

Conditional statements

Loops

5. Registration System

A simple console-based user registration system that collects user details.

Features

User registration

Input validation

Basic data storage during runtime

Concepts Used

Scanner class

String handling

Conditional statements
